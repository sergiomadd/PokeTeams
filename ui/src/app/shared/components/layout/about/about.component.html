About PokeTeams

PokeTeams is an free and open source website aimed at helping players learn and play with a pokemon team. It also allows for saving and sharing your pokemon teams, letting you have your pokepastes stored all in the same place.

Contact

If you have any issues, suggestions or questions, feel free to write an email at poketeamsweb@gmail.com or submit an issue on PokeTeams's GitHub page.
